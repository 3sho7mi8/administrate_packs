# Pakupaku
Integrate Administrate Fields and Responsive Style.
For myself.

## Usage

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'pakupaku'
```

And then execute:
```bash
$ bundle
```

## Caution

Change Uglifier harmony mode to true.

```
config.assets.js_compressor = Uglifier.new(harmony: true)
```
